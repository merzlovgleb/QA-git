Примеры:

2.1) Ошибка при редактировании задачи можно изменить стоимость на отрицательное значение

Шаги воспроизведения:

1. Зайти на сайт (https://dela-dela.cloud/)

2. Нажать на кнопку “создать задачу”

3. Ввести заголовок

4. Ввести описание задачи

5. Указать оплату и стоимость в USDT: 1 

6. Выбрать статус публикации: черновик

7. Нажать на кнопку “сохранить задачу”

8. Нажать на кнопку “редактировать”

9. Поменять оплату и стоимость USDT на -1

10. Нажать на кнопку “сохранить задачу”

11. После изменения оплаты и стоимости в USDT на отрицательное значение задача сохранилась.-баг

2.2) Ошибка: при добавлении вакансии в избранное, она не добавляется в раздел “избранные
вакансии”

Шаги воспроизведения:

1. Зайти на сайт (https://dela-dela.cloud/)

2. Переключить режим с заказчика на мастера

3. Нажать на кнопку “все вакансии”

4. Выбрать вакансию

5. Добавить вакансию в избранное 

6. Нажать на кнопку избранные вакансии

7. Вакансия не отобразилась в разделе “избранные вакансии”-баг
